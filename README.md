# Food Delivery System Receipt Generator


## Purpose of Application

This application generates a food delivery receipt.
It collects customer information, calculates subtotal,
service charge, delivery fee and displays the final receipt.


## Tech Stack

- Python
- GitHub


## How To Use

1. Run main.py
2. Enter customer information
3. Enter food details
4. Choose delivery option
5. View generated receipt
